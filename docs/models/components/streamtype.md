# StreamType

Stream type:
  - VIDEO = 1 (Video stream)
  - AUDIO = 2 (Audio stream)
  - SUBTITLE = 3 (Subtitle stream)

| Name                 | Value                |
| -------------------- | -------------------- |
| `StreamTypeVideo`    | 1                    |
| `StreamTypeAudio`    | 2                    |
